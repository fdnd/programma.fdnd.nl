Semester Static Web

Leerresultaat

De student gebruikt fundamentele principes van het web voor het ontwerpen en ontwikkelen van eenvoudige gebruiksvriendelijke interactieve toepassingen voor het web.

Inhoud

In het semester Static Web leren de studenten de fundamentele technieken van het web; HTML, CSS en JavaScript. Hiermee ontwerpen en bouwen ze statische HTML pagina's, vormgegeven met CSS en client side javascript.

In sprint 1: Your Tribe ontwerpen en bouwen studenten hun eigen online visitekaartje en werken ze samen aan een web pagine voor de squad.

De opvolgende vier sprints staan in het teken van werken aan opdrachten voor een echte opdrachtgever. Tijdens deze sprints ligt de focus op toegankelijkheid op het web, een goede gebruikservaring en het toepassen van een huisstijl in een website.

De laatste sprint, sprint 6: The Startup, werken de student een nieuw project waarin ze alles gaan toepassen wat ze het semester hebben geleerd.
