Sprint 5: Fix the Flow

Topics

  • 04/12/2023
  • search
  • filter
  • micro interaction
  • state
  • navigation
  • wireflow
  • user story

In de sprint Fix the flow leer je over navigatie systemen, hoe je informatie op een goede manier kunt presenteren en een gebruiker feedback kan geven.

De focus van deze sprint ligt op interactie en informatie architectuur van een website.

Gedragscriteria

  • M: Je past aangeboden principes en conventies op het gebied van frontend design en development toe. (1.1.2)
  • M: Je realiseert een oplossing voor een opdrachtgever en de eindgebruikers. (1.1.3)
  • C: Je kan binnen de squad ontwerpkeuzes, eigen ideeën en producten begrijpelijk verwoorden. (1.3.2)
  • P: Je schetst om gedachten en processen te verkennen. (1.4.2)
  • P: Bedenkt en implementeert eenvoudige frontend code en gebruikt daarbij de fundamentele principes van het web. (1.4.3)
  • L: Je maakt aangeboden materie eigen en gebruikt dit bij leertaken. (1.5.2)
